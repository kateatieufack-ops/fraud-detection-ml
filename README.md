\# Credit Card Fraud Detection using Machine Learning



\## Project Overview

This project implements a Random Forest model to detect fraudulent credit card transactions.



\## Dataset

The dataset contains 284,807 transactions with highly imbalanced classes.



\## Model Performance

\- Precision (Fraud): 0.96

\- Recall (Fraud): 0.74

\- F1-score: 0.84



\## Technologies Used

\- Python

\- Pandas

\- Scikit-learn

\- Joblib



\## How to Run

pip install -r requirements.txt

python train\_model.py
